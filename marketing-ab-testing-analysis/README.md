## 프로젝트 개요

**목적**  
A/B 테스트 분석 전 과정을 경험하고, 마케팅 데이터에 기본적인 통계 분석을 적용하는 것을 목표로 합니다.

**분석 관점**  
이미 설계된 실험 데이터를 바탕으로, 전환 여부에 영향을 미치는 요인을 탐색하고  
범주형 변수 중심의 통계적 검정을 통해 전환율 차이를 검증합니다.

**가설** 
• 광고 노출(ad)과 비광고 노출(psa) 간에 구매 전환율 차이가 존재하는지 확인합니다.
• 광고 노출이 구매 전환 가능성에 영향을 미칠 것이라는 가설을 검증합니다.

※ *본 프로젝트에서는 실험을 직접 설계하지 않았으며, 제공된 A/B 테스트 데이터를 기반으로 분석 질문과 가설을 정의했습니다.*


**주요 측정 지표** :`전환율 (Conversion Rate)`  
  - 구매 전환 여부(`converted`)를 기준으로 계산
  - A/B 테스트 결과 비교를 위한 핵심 성과 지표로 사용


**데이터**: 
**marketing_ab_data.csv**

| 항목 | 설명 |
|---|---|
| 데이터 출처 | [Marketing A/B Testing Dataset](https://www.kaggle.com/datasets/faviovaz/marketing-ab-testing) |
| 데이터 목적 | 광고 노출(ad)과 비광고 콘텐츠 노출(psa) 간 전환율 차이 분석 |
| 관측 단위 | 사용자 단위 (1 row = 1 user) |
| 실험 그룹 | ad (광고 노출), psa (비광고 콘텐츠 노출) |
| 주요 타깃 변수 | converted (구매 전환 여부: True / False) |
| 주요 설명 변수 | test group, total ads, most ads day, most ads hour |
| 데이터 유형 | 범주형 변수 중심, 일부 수치형 변수 포함 |

해당 데이터는 광고 노출 여부에 따른 사용자의 구매 전환 여부 차이를 분석하기 위해 설계된 A/B 테스트 데이터로, 본 프로젝트에서는 범주형 변수 중심의 분석을 수행했습니다.

## 분석 과정

1. **데이터 품질 점검** - 중복 검사 및 데이터 정제
2. **탐색적 데이터 분석** - 전환율 패턴 및 분포 확인
3. **통계적 가설 검증** - 범주형 변수에 대한 카이제곱 검정
4. **인사이트 도출** - 데이터 기반 결과 해석
<br>

---

## 분석 범위

**본 프로젝트에서 다룬 부분**
- 범주형 변수 중심의 A/B 테스트 분석
- 구매 전환 여부와 실험 그룹 간 관계 검증
- 독립성 검정을 통한 통계적 유의성 확인

**다루지 않은 부분**
- 수치형 변수에 대한 평균 차이 검정
- 다중 그룹 비교를 위한 고급 통계 기법
- 효과 크기 기반의 실용적 유의성 분석  
→ *향후 분석에서는 수치형 변수 검정 및 효과 크기 분석으로 확장 가능*
<br>

## 주요 결과

- 광고 그룹과 비광고 그룹 간 전환율 차이는 통계적으로 유의했으나(p < 0.001), 실제 차이는 0.76%p로 매우 작아 실용적 중요성은 제한적이었습니다.
- 요일 및 시간대에 따른 전환율 패턴은 명확하게 나타났으며, 주중 초반과 오후 시간대에서 전환율이 높았습니다.
- 범주형 변수 분석만으로도 광고 전략 최적화에 필요한 실행 가능한 인사이트를 도출할 수 있음을 확인했습니다.
<br>

## 도전과 해결  

- 전체 데이터가 매우 크고(ad 96%, psa 4%), 두 그룹의 불균형이 커서 통계적으로는 유의하더라도 실제 비즈니스에서 의미있지 않을 수도 있다는 결과를 얻었습니다. 따라서 p-value만으로 결과를 판단하지 않고, 전환율 차이의 크기와 실제 활용 가능성을 함께 고려해 해석했습니다.
- 또한 실험 데이터의 대부분이 범주형 변수로 구성되어 있어, 분석 범위를 넓히기보다, 핵심 질문에 집중하기 위해 수치형 변수보다 범주형 변수 중심으로 분석을 진행했습니다. 실무 경험이 없는 상태에서 복잡한 통계 기법을 무리하게 적용하기보다, A/B 테스트의 기본적인 검정 흐름을 정확히 이해하는 데 집중했습니다. 그러므로 수치형 변수에 대한 추가 검정과 효과 크기 분석은 이번 프로젝트 범위에서 제외했습니다.

## 배운 점

이 프로젝트를 통해 기본적인 A/B 테스트 과정을 이해할 수 있었습니다. 통계적 유의성이 항상 비즈니스적으로 의미 있는 차이를 의미하지는 않는다는 점을 실제 데이터로 확인했습니다. 수치형 변수로 더 탐구할 부분이 많지만, 복잡한 기법보다 데이터 특성에 맞는 검정 선택과 해석이 중요한 것을 이해하게 되었습니다.
